<h1>Movie-Recommendation-System</h1><br/>
This project is a Content-Based Movie Recommendation System that recommends movies to users based on similarity of overview, genres, keywords, cast, and crew. The system uses TF-IDF, Count Vectorization, Cosine Similarity, and NLP techniques to compute how similar two movies are.<br/><br/>

<h2>Features:</h2><br/>
1. Content-Based Filtering<br>
2. Uses movie metadata such as:<br/>
    a. Genres<br/>
    b. Overview<br/>
    c. Keywords<br/>
    d. Cast & Crew<br/>
3. TF-IDF Vectorization<br/>
4. Cosine Similarity<br/>
5. Search-Based Recommendation<br/>
6. Clean and Simple Python Code<br/>
7. Beginner-friendly structure and readable workflow.<br/>
8. Dataset Based (TMDB 5000 Movies Dataset)<br/>
9.Uses publicly available structured movie information.<br/>
